Keyboard Shortcuts

Numbers and Operations

  • 0-9 - Enter numbers
  • + - Addition
  • - - Subtraction
  • * - Multiplication
  • / - Division
  • . - Decimal point
  • Enter or = - Calculate

Control Functions

  • Escape - Clear all
  • Backspace - Delete last digit
  • Delete - Clear entry

Memory Functions

  • Ctrl + M - Store in memory
  • Ctrl + R - Recall memory
  • Ctrl + L - Clear memory
  • Ctrl + P - Add to memory

Navigation

  • Ctrl + H - Toggle history
  • Tab - Navigate buttons
  • Space - Activate focused button
