# Advanced Calculator - Final Project

## Project Overview
Build a sophisticated calculator application that goes beyond basic arithmetic, featuring scientific functions, programming modes, and advanced user interface elements.

## Features to Implement
- ✅ Basic arithmetic operations (+, -, ×, ÷)
- ✅ Scientific functions (sin, cos, tan, log, sqrt, etc.)
- ✅ Programming mode (binary, hex, octal conversions)
- ✅ Memory functions (M+, M-, MR, MC)
- ✅ History of calculations
- ✅ Keyboard input support
- ✅ Multiple themes (Standard, Scientific, Programming)
- ✅ Expression evaluation
- ✅ Error handling and validation
- ✅ Responsive design

## Learning Objectives
- Advanced JavaScript mathematical operations
- State management and data persistence
- Complex UI interactions and animations
- Regular expressions for input validation
- Event handling and keyboard shortcuts
- CSS Grid and Flexbox for complex layouts
- Error handling and edge case management
- Accessibility features for calculator interfaces

## Mathematical Functions Included
- Basic: +, -, ×, ÷, %
- Powers: x², x³, xʸ, √, ³√
- Trigonometric: sin, cos, tan, asin, acos, atan
- Logarithmic: log, ln, log₂
- Constants: π, e
- Factorials: n!
- Combinations and permutations

## Keyboard Shortcuts
- Numbers: 0-9
- Operations: +, -, *, /, %
- Functions: Enter (=), Escape (clear), Backspace (delete)
- Memory: Ctrl+M (store), Ctrl+R (recall), Ctrl+L (clear)
- History: Ctrl+H (show/hide), Up/Down arrows (navigate)

## Error Handling
- Division by zero
- Invalid mathematical operations
- Overflow and underflow
- Invalid input validation
- Malformed expressions

## How to Use
1. Open `index.html` in your browser
2. Choose calculator mode (Standard/Scientific/Programming)
3. Use mouse clicks or keyboard input
4. Access history with the history button
5. Use memory functions for storing values
6. Switch themes for different visual experiences
